# 📈 NSE Stock Screener

A mobile-friendly stock screener for Indian markets (NSE) built with Streamlit and yfinance.

## Filters Supported
- P/E Ratio (range)
- Market Cap (minimum, in ₹ Crore)
- Return on Equity / ROE (minimum %)
- Sales Growth YoY (minimum %)
- Profit Growth YoY (minimum %)
- Sector filter
- Growth vs Industry Average comparison

## 🚀 Running Locally

### Step 1 — Install Python
Download Python 3.10+ from https://python.org

### Step 2 — Install dependencies
```
pip install -r requirements.txt
```

### Step 3 — Run the app
```
streamlit run app.py
```

## ☁️ Deploy Free on Streamlit Cloud

1. Create a free account at https://streamlit.io/cloud
2. Push this folder to a GitHub repository
3. In Streamlit Cloud → "New App" → connect your GitHub repo
4. Set the main file as `app.py`
5. Click Deploy!
